# BlueStreamline App Integration - Student Formula Project

## Project Overview
This project is a dedicated module integrated into the Universitatea Transilvania din Brașov mobile ecosystem. It was designed specifically for BlueStreamline, the first Romanian team to compete in the Formula Student international engineering competition.

The application serves as a central hub for team management, recruitment, and public relations, bridging the gap between the engineering team and the student body.

## Technical Implementation
Developed as part of a 4-member team, I contributed to the full lifecycle of this section, from conceptual design to deployment.

* **Frontend:** Built with JavaScript and React, ensuring a responsive and dynamic user experience.
* **Styling:** Utilized Tailwind CSS and standard CSS to implement a high-contrast "dark mode" aesthetic that aligns with the racing team's branding.
* **Integration:** Responsible for creating the user interface and ensuring seamless integration with the backend API and database for real-time data handling (recruitment forms, team status).

## Key Features
* **Recruitment Management:** A full-cycle module allowing students to apply and check their interview status via a dedicated dashboard.
* **Team Hierarchy & Crew:** Interactive display of the organizational structure (Combustion vs. Electric vehicle leads).
* **Interactive History:** A chronological look at the team’s evolution and car models (BS 09).
* **Testimonials:** A section for members to share hands-on engineering experiences.

## Repository Note
**Code Availability:** This repository contains only selected modules and specific code snippets to demonstrate UI/UX logic and integration patterns. The complete project remains private due to university internal policies and team data protection.

## Team Process
Working in a team of four, we followed an agile-like process:
* **Design:** Mapping the user flow for recruitment.
* **Implementation:** Splitting tasks between UI components and backend connectivity.
* **Testing:** Ensuring data persistence and responsive layouts across different mobile screen sizes.
